# Okamoto Security Labs — MVP

🚀 Landing page da **Okamoto Security Labs** — uma plataforma de análise de logs com IA que gera relatórios **executivos e técnicos em PDF**, em minutos.

🔗 Acesse a página: [okamoto-security-labs](https://gustavo89587.github.io/okamoto-security-labs/)

## ⚡ Funcionalidades
- Upload de logs (`apache`, `nginx`, `syslog`, `wazuh`)
- Normalização automática de eventos
- Classificação de severidade
- Geração de relatórios PDF
- Interface simples e responsiva em **Bootstrap 5**
